2 Calcular la Correlacin

In[6]:
# Correlacin entre matemticas y lectura
r, p_value = pearsonr(df["math score"], df["reading score"])

print(f"Coeficiente de correlacin: {r:.4f}, Valor p: {p_value:.4f}")

# Visualizar con un heatmap
plt.figure(figsize=(6,4))
sns.heatmap(correlaciones, annot=True, cmap="coolwarm", fmt=".2f")
plt.title("Matriz de Correlacin")
plt.show()

Coeficiente de correlacin: 0.8176, Valor p: 0.0000

Interpretacin:

  • Si r est cerca de 1 o -1, hay una fuerte relacin.
  • Si p-value < 0.05, la correlacin es estadsticamente significativa.

3 Ajustar un Modelo de Regresin Lineal

Pregunta: Podemos predecir la calificacin en escritura con la calificacin en matemticas?

In[7]:
# Definir variables
X = df[["math score"]]  # Variable independiente
y = df["writing score"]  # Variable dependiente

# Crear el modelo
modelo = LinearRegression()
modelo.fit(X, y)

# Obtener coeficientes
b0 = modelo.intercept_
b1 = modelo.coef_[0]

print(f"Ecuacin de regresin: Y = {b0:.2f} + {b1:.2f}X")

Ecuacin de regresin: Y = 14.89 + 0.80X

4 Hacer Predicciones

In[8]:
# Hacer predicciones
df["Prediccin escritura"] = modelo.predict(X)

# Graficar la regresin
plt.figure(figsize=(8,5))
sns.scatterplot(x=df["math score"], y=df["writing score"], label="Datos Reales")
sns.lineplot(x=df["math score"], y=df["Prediccin escritura"], color="red", label="Regresin")
plt.xlabel("Calificacin en Matemticas")
plt.ylabel("Calificacin en Escritura")
plt.title("Regresin Lineal: Matemticas vs Escritura")
plt.legend()
plt.show()

Interpretacin:

  • La lnea roja muestra la tendencia.
  • Si los puntos estn cerca de la lnea buena prediccin.
  • Si estn dispersos el modelo no es bueno.

5 Evaluar el Modelo

In[9]:
from sklearn.metrics import mean_squared_error, r2_score

# Mtricas de rendimiento
mse = mean_squared_error(y, df["Prediccin escritura"])
r2 = r2_score(y, df["Prediccin escritura"])

print(f"Error cuadrtico medio (MSE): {mse:.4f}")
print(f"Coeficiente de determinacin (R): {r2:.4f}")

Error cuadrtico medio (MSE): 82.0670
Coeficiente de determinacin (R): 0.6442

Interpretacin:

  • MSE bajo Mejor modelo (menor error).
  • R cerca de 1 El modelo explica bien la variabilidad.
